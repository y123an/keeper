# keeper

## General Info
- if is a note taking app that is used to keep track of you work or other activies

## Technologies
- React.js
- Material UI

## Get started

First,to install the packages by running
```bash
- npm i
```
setup the server
```bash
- npm start
```
then run localhost:3000 in the browser

## features
- add notes
- delete notes
- update notes
- read notes
